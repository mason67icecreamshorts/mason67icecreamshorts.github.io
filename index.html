Mini Web DAW
Open DAW

Lightweight Web DAW

Create beats and arrangements in your browser. Sample loading, step sequencer, per-track pan, reverb & delay, pattern saving, MIDI input and export to WAV.

Open the DAW

Features

  • 16-step sequencer with 8 tracks
  • Load your own samples per track
  • Per-track pan, volume, mute, reverb & delay sends
  • Save/load patterns (localStorage)
  • Export mix to WAV
  • Basic Web MIDI support
Made with Tone.js — local only, no server processing.
